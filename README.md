# Central Utilities Hub

A performance-optimized, lightweight suite of client-side web applications built using vanilla frontend technologies.

## 🚀 Live Production URL
Visit the live portal here: [7-tarun.github.io](https://7-tarun.github.io)

## 🛠️ Hosted Applications

1. **Eye Rest Monitor:** A digital wellbeing tool enforcing the 20-20-20 rule with custom intervals and sound notifications to reduce digital eye strain.
2. **Advanced HealthCalc:** A client-side demographic-aware fitness tracking dashboard featuring BMI and strict U.S. Navy standard Body Fat calculators.

## 💻 Tech Stack
* **Architecture:** Semantic HTML5, Mobile-First CSS3 Grid/Flexbox
* **Logic:** Vanilla JavaScript (ES6+), LocalStorage State Management
* **Hosting:** GitHub Pages
